As we know, WeCom Integration enables businesses to engage with customers on WeChat through their WeCom enterprise account. This is made possible by the WeChat Customer Service APIs provided by WeCom.

What is Wechat Customer Service Account?

[Customer Service Account](https://developer.work.weixin.qq.com/document/path/94683) - A Customer Service Account is a channel that allows WeCom enterprises to have conversations with WeChat customers. Currently, a company can add up to 5,000 Customer Service Accounts. These accounts can be created either through the WeCom organizationâ€™s Admin Panel or via the WeCom API.

### SDK Implementation:

We have two apis of Customer Service Account - Get Customer Service Account List and Create Customer Service Account.

###### 1\. Get Wecom CSAccounts:

To know about Customer Service Account - [CS Account Document](https://learn.zoho.in/portal/zohocorp/knowledge/manual/available-integrations/article/wecom-customer-service-accounts)

Requirements:

1\. WecomCorpId - Wecom Corporation Id

2\. Range Limit

```
import java.util.List;

import com.zoho.im.client.model.IMEntityList;
import com.zoho.im.client.model.IMWeComCustomerServiceAccount;
public List&lt;IMWeComCustomerServiceAccount&gt; getCustomerServiceAccounts(
                String weComCorpId, int offset, int limit) throws Exception {

            // Build query
            IMWeComCustomerServiceAccount.IMWeComCustomerServiceAccountQuery query =
                    new IMWeComCustomerServiceAccount.IMWeComCustomerServiceAccountQuery().weComCorpId(weComCorpId);
            query.range(offset, limit);
            // Execute query
            IMEntityList&lt;IMWeComCustomerServiceAccount&gt; accounts =
                    ZohoIM.getInstance(IM_ORG_ID, SERVICE_ORG_ID).getIntegrationCustomAPI(USER_ID).getWeComCustomerServiceAccounts(query);

            // Print account details
            for (IMWeComCustomerServiceAccount account : accounts.getEntities()) {
                System.out.println("Account: " + account.getAccountName());
                System.out.println("OpenKfId: " + account.openKfId());
                System.out.println("Has Manage Privilege: " + account.getHasManagePrivilege());
                System.out.println("Profile Picture: " + account.getProfilePicUrl());
                System.out.println("---");
            }

            return accounts.getEntities();
     }
```

Response:

1.  Wecom Customer Service Account - account Name

2.  Open KFID - Customer Service Account's Unique Id
3.  hasManagePrivilege  - is Zoho having manage privilege for the specific Customer Service Account
4.  ProfilePicUrl - Profile Picture of Customer Service Account

###### 2\. Wecom Customer Service Account Creation

Requirements:
1\. WeComCorpId - Corp ID that we got from parentaccounts

2\. Customer Service Account Name - Name that need to be given for Customer Service Account

3\. Attachment Id - Attachment ID that is associated with IM. Image that is already uploaded with IM.

```
import com.zoho.im.client.model.IMWeComCustomerServiceAccount;

public String createCustomerServiceAccount(String weComCorpId, String accountName, Long profilePicAttachmentId)
            throws Exception {

        // Build customer service account
        IMWeComCustomerServiceAccount.IMWeComCustomerServiceAccountBuilder builder =
                new IMWeComCustomerServiceAccount.IMWeComCustomerServiceAccountBuilder()
                        .weComCorpId(weComCorpId)
                        .customerServiceAccountName(accountName)
                        .attachmentId(profilePicAttachmentId);

        // Create the account
        IMWeComCustomerServiceAccount.IMWeComCustomerAccountCreationResponse response =
                ZohoIM.getInstance(IM_ORG_ID, SERVICE_ORG_ID).getIntegrationCustomAPI(USER_ID).createWeComCustomerServiceAccount(builder);

        String openKfId = response.getOpenKfId();

        System.out.println("OpenKfId: " + openKfId);

        return openKfId;
 }
```
