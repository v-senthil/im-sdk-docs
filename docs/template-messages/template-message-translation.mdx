Get Single Template message Translation :-

## Note :- To use this translation feature, use [this sdk](/portal/zohocorp/manual/im-developers-corner/article/changelog-1#_Toci7zfcxr8edll) in IM

```
  IMCannedMessageTranslation translation = ZohoIM.getInstance(IM_ORG_ID, IM_ORG_ID)
                .getCannedMessageTranslationAPI(USER_ID)
                .get(CANNED_MESSAGE_ID, TRANSLATION_ID);
```

Get Template Message Translation List :-

```
IMEntityList&lt;IMCannedMessageTranslation&gt; imTranslations = ZohoIM.getInstance(IM_ORG_ID, IM_ORG_ID)
                .getCannedMessageTranslationAPI(USER_ID)
                .getList(CANNED_MESSAGE_ID, null);
```

Create Template Message Translation :-

## 1\. Create a Template Message Translation with Body Message

```
//Create Translation Object
IMCannedMessageTranslation translationEnglish = new IMCannedMessageTranslation
        .Builder("en", "Translation in english", null).build();

//Save Translation Object in IM
IMCannedMessageTranslation translation = ZohoIM.getInstance(IM_ORG_ID, SERVICE_ORG_ID)
                .getCannedMessageTranslationAPI(USER_ID)
                .create(CANNED_MESSAGE_ID, translationEnglish);
```

2\. Create a Template Message Translation with Body Message and header

```
Builder translationProperty = new IMTemplateItems.Builder();
translationProperty.setHeader(new IMTemplateHeader(IMTemplateHeaderType.TEXT, "hello header"));

//Create Translation Object
IMCannedMessageTranslation translationEnglish = new IMCannedMessageTranslation
        .Builder("en", "Translation in english", translationProperty.build()).build();

//Save Translation Object in IM

                .getCannedMessageTranslationAPI(USER_ID)
 .create(CANNED_MESSAGE_ID, translationEnglish);
```

Update a Template Message Translation

Constraints in Editing a Template Message Translation:

1\. The language of the template translation cannot be changed.

2.Template Translation of Status - DELETION\_IN\_PROGRESS, PENDING, PENDING\_DELETION cannot be edited.

3.An Approved template can be edit only once per 24 hours.

## 1\. Update a Template Message Translation with Body Message

```
IMCannedMessageTranslation translationEnglish = new IMCannedMessageTranslation
        .Builder("en", "Translation in english", null)
          .setId(TRANSLATION_ID)
          .build();

//Update Translation Object in IM
IMCannedMessageTranslation translation = ZohoIM.getInstance(IM_ORG_ID, SERVICE_ORG_ID)
                .getCannedMessageTranslationAPI(USER_ID)
 .update(CANNED_MESSAGE_ID, translationEnglish);
```

2\. Update a Template Message Translation with Body Message and header

Delete a Template Message Translation

```
boolean deletedCannedMessage = ZohoIM.getInstance(IM_ORG_ID, IM_ORG_ID)
                .getCannedMessageTranslationAPI(USER_ID)
                   .delete(CANNED_MESSAGE_ID, TRANSLATION_ID);
```

Note :-

1.  If the primary language translation is deleted, then the whole canned message will be deleted along with all the other translation.
2.  If a translation is deleted with a language, then whatsapp will not allow you to create the same translation with the same language in the same canned message entity for 4 weeks. In this case, if needed to create the translation with that language, then another canned message can be created and that translation can be added there.
