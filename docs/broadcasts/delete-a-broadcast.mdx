---
title: Delete a broadcast
sidebar_position: 4
description: Only the status of the broadcast will be deleted. No data will be deleted / lost.
---
```
ZohoIM.getInstance(IM_ORG_ID,SERVICE_ORG_ID)
        .getBroadcastAPI(USER_ID)
        .delete(BROADCAST_ID);
```

Only the status of the broadcast will be deleted. No data will be deleted / lost.
