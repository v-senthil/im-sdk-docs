### Organization

    The "Organization" in IM serves as the primary entity which represents a company. A single organization can be created in IM for each company, enabling them to add channels and carry out various other IM-related tasks. Zoho IM is in IAM Service Org Account model and an IM Service Org account needs to be created to get started.

OAuth Scope: ZohoIM.organizations.ALL

Save the created IM organization ID as it is required to invoke all other IM APIs.
