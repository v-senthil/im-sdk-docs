The Payment Statement API is designed to provide the payment history in IM for a specified Time duration(Maximum 31 days).

Payment Statement API provides records of following events:

1\. Credits Purchase,

2\. Debit of IM Credits,

3\. Renewal of IM Credits,

4\. UnSuccessful Debit of IM Credits,

5\. Pending Clearence(If any)

It will be helpful to check how credits were spent in a given duration. And It will provide detailed information regarding the reason for a particular debit transaction.

Note:

1.  Data will be available from March 22,2023 in Live servers.
2.  Maximum duration provided will be 30 days.
3.  For pagination of records, the maximum limit is 100 records. By default Limit will be 30 and from will be 0.
4.  If No Transaction Type filter given all Transaction type records will be fetched.
5.  serviceName filter will be helpful provide the results of particular ZohoService based transaction in Bundled Integration. If it is not provided, then will be providing all the transactions in the particular bundled service like CRM Plus or Zoho One.
