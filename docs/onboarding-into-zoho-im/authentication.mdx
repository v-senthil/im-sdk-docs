1.  Zoho Instant Messaging supports:

1.  ISC Signature
2.  User's IAM Current Ticket
3.  Org-Based OAuth Token
4.  User-Based OAuth Token

2.  Most of the APIs support both User-Based OAuth Token and Org-Based OAuth Token.
3.  We expect service, service organization ID and Zoho IM Org ID in API headers for Authorization.
