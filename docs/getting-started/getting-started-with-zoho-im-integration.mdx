1.  Contact IM Team with the Use-case.
2.  ZohoIM can be used in two Perspectives

1.  Native Integration for Customers. Each customer can configure their own Number.
2.  To Send Product Notifications(Similar to Trans-mail, Zoho Classes)

3.  Provide your:

1.  Service Type as in com.zoho.accounts.AccountsConstants.ServiceType
2.  Org Type as in com.zoho.accounts.AccountsConstants.OrgType
3.  Service Name to be configured in scoped-services
4.  Store ID of your service

4.  IM team will add a configuration in our end with your service details so that you can start using IM.
5.  After a basic testing in csez, we shall start the process of payment add-ons. IM Team will request an IM add-on for your service to the store team.
6.  Real time notifications through kafka [link](/portal/zohocorp/team/zohoim/manual/sdk-reference/article/real-time-notifications)
