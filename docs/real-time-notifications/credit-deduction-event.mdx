The event will be triggered when some amount of credits has to be deducted. This event will only be pushed to serviced who have their own credit system.

Kindly contact the IM Team if your service wishes to use this event.

```
public void imCreditsUsageEvent(IMEventMeta eventMeta, IMCreditDeductionEvent event)
```
