The event will be triggered when a new message is received.

```
public void messageAdded(IMEventMeta eventMeta, IMMessageAddedEvent messageAddedEvent)
```

IMEventMeta - Reference [link](/portal/zohocorp/manual/sdk-reference/article/introduction-8#_Tocx9f5qofl8hsz)
