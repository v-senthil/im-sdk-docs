### IMSessionStatus

1.  Created - When a new incoming message and a new session is created
2.  Open - If a session already exists, when a new message is received for that session.
3.  On Hold - If an agent takes on a session.
4.  On Progress - When an agent responds to a session.
5.  Ended - When an agent terminates a session.
6.  Blocked - When an agent prohibits a session.
