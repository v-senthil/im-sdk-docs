Overview

A WhatsApp Pricing Calculator(WPC) is typically a tool designed to estimate the costs associated with using WhatsApp's Business APIs for communication. WhatsApp charges businesses based on conversation-based pricing, which depends on the type of conversation, the number of messages sent, and the region of the recipient.

![Illustration](/img/inline-images/efc/whatsapp-pricing-calculator/image-6.png)

Initialization

Before using the SDK, an object named ZOHOIM must be initialised on the window object as follows :

```
window.ZOHOIM = window.ZOHOIM || {};
```

Setting Required Properties

To render the WPC, certain properties must be set within the window.ZOHOIM object. These include:

SDK Script for WPC

```
&lt;html&gt;

&lt;head&gt;
  &lt;title&gt;WPC</title>
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.7.1/jquery.min.js"></script>
  &lt;script&gt;
  function initWhatsAppEmbeddedSdk() {
        window.ZOHOIM = window.ZOHOIM || {};
        window.ZOHOIM.widgetProps = {
                isBuyCreditsBtnNeeded: false,
                isNeedTitle: true,
                handleBuyCredits: () => {
                    alert('hi')
                },
                activeTemplate: 'utility'
            },

            jQuery.ajax({
                url: `https://im.zoho.com/api/v1/public/sdk/efc?widgetType=whatsAppCalculator`,

                type: "GET",
                contentType: "application/json",
                success: function(response) {
                    var { data } = response;
                    var newScript = document.createElement("script");
                    let parser = new DOMParser();
                    let doc = parser.parseFromString(data, 'text/html');

                    let scriptElement = doc.querySelector('script[type="text/javascript"]');
                    var scriptStr = "";
                    if (scriptElement) {
                        scriptStr = scriptElement.textContent;
                        console.log(scriptStr);
                    } else {
                        console.log("No matching &lt;script&gt; tag found.");
                    }
                    newScript.appendChild(document.createTextNode(scriptStr));
                    document.head.append(newScript);
                },
                error: function() {
                    console.error("Failed to load the WhatsApp Pricing Calculator script.");
                }
            });
    }

function loadWpc() {
    initWhatsAppEmbeddedSdk();
}
  </script>
</head>

&lt;body&gt;
  <div id="whatsApp-pricing-calculator"></div>
  <button onclick="loadWpc()">Load WPC</button>
</body>

</html>
```
