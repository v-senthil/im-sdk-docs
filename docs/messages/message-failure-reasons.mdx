```
IMMessage message = ; // get IMMessage object using getSessionMessages or getMessages API
String messageFailureCode = message.getMeta().get(IMMessageMeta.FAILED_REASON);
IMMessageStatusErrorCode messageStatusErrorCode = IMMessageStatusErrorCode.valueOf(messageFailureCode);
String statusDescription = messageStatusErrorCode.getStatusMessage();
```

## WhatsApp Integration

### INVALID\_RECEIVER

It could be one of the following reasons.
![Illustration](/img/inline-images/messages/message-failure-reasons/1726110812933.png)

We will not be receiving the exact reason why the message failed. When checked with meta, below was their reply,

If a marketing template message is not delivered to a given user due to the limit, Cloud API will return error code 131026. Note, however, that these error codes cover a wide range of issues that can result in non-delivery of a message, and for privacy reasons, we will not disclose if in fact the message was not delivered due to the limit.
