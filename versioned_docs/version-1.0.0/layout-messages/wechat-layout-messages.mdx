### Constraints :

1) URL buttons :

-   Button Limit - 1
-   Message Character Limit - 20
-   Action Character Limit - 1000

2) Reply Buttons :

-   Button Limit - 8
-   Message Character Limit - 20

3) Providing one button (either url or reply) is mandatory

4) Description is supported with a char limit of 512 (mandatory)

5) Title is supported with a char limit of 512 (mandatory)

6) Subtitle is not supported

7) Attachments are not supported

#### Send Normal Layout Messages with URL buttons :

```
private static final Long SESSION_ID = 435345l;
    public static final Long IM_ORG_ID = 74203785L;     //Created by IMOrganization Creation
    public static final Long SERVICE_ORG_ID = 74199498L;     //Zoho ServiceOrg ID or AppAccount ID
    public static final Long USER_ID = 69718860L;          //Zoho User ID
    public static void main(String []args) {
        try {
            IMLayoutMessageBuilder.Builder builder = new IMLayoutMessageBuilder.Builder(SESSION_ID);

            // set description
            builder.setDescription("This is description");
            // set title
          builder.setTitle("This is title", null);

            // set buttons
            IMLayoutButton buttonOne = new IMLayoutButton.Builder(IMLayoutButtonElementType.REDIRECT, "visit", "https://zoho.com/3").build();

            List&lt;IMLayoutButton&gt; buttons = new ArrayList&lt;&gt;();
            buttons.add(buttonOne);

            IMLayoutMessageButton layoutButton = new IMLayoutMessageButton.
                    Builder(IMLayoutButtonStyle.URL_BUTTON,
                    IMLayoutButtonType.BUTTON, buttons)
                    .build();
            builder.setButtons(layoutButton);

            // Send Message
            IMMessage message = ZohoIM.getInstance(IM_ORG_ID, SERVICE_ORG_ID)
                    .getMessageAPI(USER_ID)
                    .layoutMessage(builder.build());
            System.out.println(message.getId());
        }catch (Exception e) {
            System.out.println(e.getMessage());
        }
 }
```
