### postman collection : [link](https://documenter.getpostman.com/view/18635004/2s935vkfHz#102cfe39-b687-42e9-a06e-0a2ec43a0449)

### Constraints :

1) Description is mandatory

2) Description limit - 640 Characters

3) Title and Subtitle are not supported

4) Providing one button in mandatory

5) URL buttons :

-   Button Limit - 3
-   Message Character Limit - 20
-   Action Character Limit - 1000

6) Reply Buttons :

-   Button Limit - 13
-   Message Character Limit - 20

7) Attachments are not supported
