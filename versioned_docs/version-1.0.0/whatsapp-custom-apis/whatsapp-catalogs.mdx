### What is WhatsApp Catalog ?

Customers can showcase and share their products and services with end users for them to browse items and add to a cart without leaving the chat which is displayed on their business profile. End consumers will view and purchase a product based on the needs. Ask for any information on a product or send an order. To setup WhatsApp commerce platform, we need to do following steps:

1.  Create a Catalog
2.  Link catalog with the WhatsApp business account.
3.  Enable Cart and Catalog for a specific mobile number.

For this, we have exposed an APIs.

Example of WhatsApp Catalog:

### SDK for Get Catalog connected with WABA

Request:

```
IMBasicObject catalogDetails = ZohoIM.getInstance(IM_ORG_ID, SERVICE_ORG_ID)
                            .getIntegrationCustomAPI(ZUID)
                         .getWabaCatalog(wabaId);
```

Response

```
{
    "data": [
        {
            "name": "Zylker Catalog",
            "id": "1751710589558678"
        }
    ],
    "kind": "single.json"
}
```

### SDK for Connect catalog with WABA

###
[Using this API, we can connect a catalog with a WhatsApp business account.](https://documenter.getpostman.com/view/18635004/2s935vkfHz#48a3dd8c-e043-40b0-911e-bdb90d8f2393)

[Request:](https://documenter.getpostman.com/view/18635004/2s935vkfHz#48a3dd8c-e043-40b0-911e-bdb90d8f2393)

```
Long imOrgId = 5467487L;
Long serviceOrgId = 87534345L;
Long userId = 832748234L;
String catalogId= "2397423478";
IMCatalog catalog = new IMCatalog(catalogId);
String wabaId = "2342347892";
boolean isLinked = ZohoIM.getInstance(imOrgId, serviceOrgId)
                            .getIntegrationCustomAPI(userId)
                         .linkCatalogWithWaba(wabaId, catalog);
```

### SDK for Disconnect catalog with WABA

Using this API, we can disconnect a catalog from WhatsApp business account

[Request:](https://documenter.getpostman.com/view/18635004/2s935vkfHz#c29eb11a-1588-4719-acca-7a6dc7fb074b)

```
Long imOrgId = 5467487L;
Long serviceOrgId = 87534345L;
Long userId = 832748234L;
String catalogId= "2397423478";
IMCatalog catalog = new IMCatalog(catalogId);
String wabaId = "2342347892";
boolean isUnLinked = ZohoIM.getInstance(imOrgId, serviceOrgId)
                .getIntegrationCustomAPI(userId)
                .unLinkCatalogWithWaba(wabaId, catalog);
```

### SDK to Apply Cart Settings to WhatsApp

Enable/Disable Cart: When the cart is disabled, customers can see products and their details, but all cart related buttons will not appear in any view.

Enable/Disable Catalog: When the catalog is disabled, the storefront icon and catalog-related buttons will not appear in any views and the catalog preview with thumbnails will not appear in the business profile view.

Using this API, we can enable or disable cart and catalog for a specific WhatsApp phone number.

Request:

```
Long imOrgId = 5467487L;
Long serviceOrgId = 87534345L;
Long userId = 832748234L;
Long phoneNumberId = 893745387L;
IMCommerceSettings imCommerceSettings = new IMCommerceSettings(true, true); // Params can be vary based on required in case user wishes to disable cart they can create object like this { new IMCommerceSettings(false, true); }, similary it can be vary.
boolean isCartSettingsApplied = ZohoIM.getInstance(imOrgId, serviceOrgId)
                .getIntegrationCustomAPI(userId)
                .applyCartSettingsToBusinessPhoneNumber(phoneNumberId, imCommerceSettings);
```
