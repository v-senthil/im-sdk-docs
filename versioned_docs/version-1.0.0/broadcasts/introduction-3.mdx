A broadcast message is a message that is sent to multiple recipients simultaneously. Once you create a broadcast, we will instantly start sending the messages after we provide basic broadcast details as a response.

Postman collection: [link](https://documenter.getpostman.com/view/18635004/2s935vkfHz#17509c7b-f824-4743-ae53-a67fb0afa680)

OAuth Scopes: ZohoIM.channels.ALL, ZohoIM.messages.ALL

Note:

1.  Once a broadcast is created, the broadcast can not be stopped or deleted.
2.  Only completed broadcasts can be deleted.
3.  We still haven't supported scheduled broadcast (will be available in upcoming days).

### Whatsapp Broadcast

1.  For now, Broadcast is supported only on WhatsApp.
2.  Both custom body messages and simple body messages are supported.
3.  Message will be sent using a valid channel\* and a valid template Message\*

\*valid channel

1.  Channel should be active
2.  Channel should not be deleted
3.  Channel should be a WhatsApp integrated channel

\*valid template Message

1.  Template message should be active and approved by whatsapp
