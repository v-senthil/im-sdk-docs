### Note: We advise service teams to get the Template Category and Languages supported via the API we provide.

###

### Get Template Message Supported Categories

[Postman documentation link](https://documenter.getpostman.com/view/1858532/Tzscomr8#3e2d15a2-8a4b-40ad-926f-15a7da220f10)

1.  Utility
2.  Marketing
3.  Authentication

Note: If the category is Authentication, the creation flow will be changed on May 29th. Thus we have removed Authentication category for a time period. We will work on it and provide you updates as soon as possible.

1.  UTILITY: Utility templates relate to a specific, agreed-upon transaction and accomplish one of the following:

1\. Confirm or suspend a transaction

2\. Provide an update or reminder post-transaction.

Examples: order confirmations, account updates, receipts, appointment reminders, billing

2.  MARKETING: Marketing templates are our most flexible â€“ they do not relate to a specific, agreed-upon transaction and instead may relate to the business and/or its products/services. These templates may include promotions or offers; welcoming / closing messages; updates, invitations or recommendations; or requests to respond or complete a new transaction.

Examples: welcome messages, newsletters, offers, coupons, catalogues, new store hours

Any template that has a mix of utility and marketing content will be classified as a marketing template.

3.  AUTHENTICATION: Authentication templates enable businesses to authenticate users with one-time passcodes (usually 4-8 digit alphanumeric codes), potentially at multiple steps in the login process (e.g., account verification, account recovery, integrity challenges).

Examples: one-time password, account recovery code

For More details on Template Categories - [link](https://developers.facebook.com/docs/whatsapp/updates-to-pricing/new-template-guidelines)

```
IMBasicObject data = ZohoIM.getInstance(IM_ORG_ID, SERVICE_ORG_ID)
        .getIntegrationCustomAPI(USER_ID)
        .invokeAPI(IntegrationCustomAPIEntity.WHATSAPP_TEMPLATETAGS ,Method.GET,null);
```

### Get Template Message Created Languages

This will provide only the template languages created in your IM organization.

```
boolean isCreatedOnlyTemplateLanguages = true;
IMBasicObject data = ZohoIM.getInstance(IM_ORG_ID, SERVICE_ORG_ID)
        .getIntegrationCustomAPI(USER_ID)
        .getTemplateSupportedLanguages(isCreatedOnlyTemplateLanguages);
```
