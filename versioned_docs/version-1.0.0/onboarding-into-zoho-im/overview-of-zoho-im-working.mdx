title: Overview of Zoho IM working
sidebar_position: 7
description: All other entities will be built upon these mentioned main entities.
---
1.  A single organization can be created in IM for each company, enabling them to add channels. Zoho IM is in IAM Service Org Account model and an IM Service Org account needs to be created to get started.
2.  For each organization there can be multiple channels depending on the use cases of the customer. Channel can be created in Zoho IM with some configuration parameters. These configuration parameters may vary between integration services.
3.  Within each channel, multiple sessions can be established for communication between end users and agents. Each end user will have their own session.
4.  Each session will be connected to a contact in IM.
5.  These sessions consist of multiple messages, which can be text-based, images, videos, audio recordings, or any other type of media exchanged between users.

All other entities will be built upon these mentioned main entities.
