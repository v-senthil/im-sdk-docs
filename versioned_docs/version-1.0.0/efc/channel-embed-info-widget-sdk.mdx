Overview

-   The Channel Embed Info Widget SDK enables developers to seamlessly embed channel information into their UI by providing necessary details.
-   Using this SDK, you can easily display the necessary information to embed key communication elements for a channel, such as link, QR and Button for various channels, such as WhatsApp, Telegram, WeChat, and others, making it adaptable for different types of integrations.

Key Benefits

-   Ease of Integration : Developers can quickly set up and render the Channel Embed Info widget by simply configuring the required properties in window.ZOHOIM.
-   Multi-Channel Support : The widget supports various messaging channels (e.g., WhatsApp, Telegram, WeChat, etc.), allowing flexible integration across multiple platforms.
-   Customization : The widgetProps object allows for customization based on specific channel details.

Initialization

Before using the SDK, an object named ZOHOIM must be initialised on the window object as follows :

```
window.ZOHOIM = window.ZOHOIM || {};
```

Setting Required Properties

To render the widget, certain properties must be set within the window.ZOHOIM object. These include:

Getting the SDK Script and Rendering It:

```
window.ZOHOIM = window.ZOHOIM || {};

window.ZOHOIM.widgetProps = {
     channel: "WHATSAPP",
     channelName: "Test Channel",
     universalLink: "https://wa.me/14694059939"
 };
```
