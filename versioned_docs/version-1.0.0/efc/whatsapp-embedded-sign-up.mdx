#### Initialize Widget

```
function initWhatsAppEmbeddedSdk() {
 jQuery.ajax({
      url : `/api/v1/sdk/efc?orgId=60349982&widgetType=whatsAppEmbeddedSignup`,
      type : "GET",
      contentType : "application/json",
      headers : {
       'X-ZCSRF-TOKEN' : getCSRFCookieName() + '='+ getCSRFCookie()
      },
      success : function(response){
     var { data } = response;
     var newScript =        document.createElement("script");

     var scriptStr = data.replace("<script type='text/javascript'>", "").replace("</script>", "");
           newScript.appendChild(document.createTextNode(scriptStr));
           document.head.append(newScript);
      },
      error : function() {
          //error callback
      }
    });
}
```

After initializing the Widget by the above API, you can add your own button.
For that pass onClick as ZOHOIM.launchWhatsAppEmbeddedSignup(props, callback);

```
// orgHeader - scope based on the service
const props = { orgId: '', orgHeader: '', userZuId: '', csrfToken: '', csrfParamName: '' };
const successCallback = function(response) {
 const { wabaStatus, numberStatus } = response;
 // Can handle based on the response
}
const failureCallback = function(response) {
     const { wabaStatus, numberStatus } = response;
     // Can handle based on the response
}
window.ZOHOIM.context = '/support'; // based on service handling for crmplus.
window.ZOHOIM.launchWhatsAppEmbeddedSignup(props, successCallback, failureCallback);
```

#### Callback Status:

```
{
  "wabaStatus": {{value}},
  "numberStatus": {{value}}
}
```

Possible wabaStatus values:

NONE - No WABA present

PENDING - WABA added and approval pending

APPROVED - WABA added and approved

REJECTED - WABA added and rejected approval

Possible numberStatus values:

PRESENT - at least one phone number is added to WABA

ABSENT - no phone numbers are added to WABA

### Possible errors

1.  WABAID\_DIFFERS

1.  The mapped WABA in an organization is different from the WABA customer choose during embedded sign up. This error will occur for services which does not have multi WABA support.
