### +1 555 WhatsApp Business Phone numbers

In order to send Message via +1 555 Business Phone Numbers provided by WhatsApp, the phone number  display name must be reviewed and approved.

If the display name of the phone number +1 555 has no status. Kindly re-submit the display name entirely and check if this reset the process.

To resubmit another Display name using the WhatsApp Business Manager UI. Please use the guide below to make sure your desired Display Name meets our requirements: [https://www.facebook.com/business/help/338047025165344#display-name-guidelines](https://www.facebook.com/business/help/338047025165344#display-name-guidelines)

1\. In WhatsApp Manager navigate to the "Phone Numbers" tab
2\. Click on the Pencil button next to the respective Display Name that you would like to edit
3\. Type in the desired Display Name and Click "Next"
