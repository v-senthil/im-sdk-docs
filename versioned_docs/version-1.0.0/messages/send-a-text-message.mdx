```
//construct message object
IMTextMessage textMessage = new IMTextMessage("Hey there!");
IMMessage messageToSend = new IMMessage.Builder(textMessage)
      .setReplyToMessageId(replyToMessageId) // Optional, if reply to message need to be sent
      .build();

      //send message through IM
IMMessage sentMessage = ZohoIM.getInstance(IM_ORG_ID, SERVICE_ORG_ID)
      .getMessageAPI(USER_ID)
      .sendMessage(IM_SESSION_ID, messageToSend);
```
