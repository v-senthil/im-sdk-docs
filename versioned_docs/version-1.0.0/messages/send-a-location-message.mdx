```
IMLocationMessage location = new IMLocationMessage("Here is our office location","12.721821", "31.37231");  //message, latitude, longitude

// Construct IMMessage object with location data
IMMessage messageToSend = new IMMessage.Builder(location)
      .setDirection(Direction.OUT)
      .build();

//send attchment message through IM
IMMessage sentMessage = ZohoIM.getInstance(IM_ORG_ID, SERVICE_ORG_ID)
      .getMessageAPI(USER_ID)
      .sendMessage(IM_SESSION_ID, messageToSend);
```
