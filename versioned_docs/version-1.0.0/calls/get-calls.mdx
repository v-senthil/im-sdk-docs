### Get single Call

```
IMCalls call = ZohoIM.getInstance(IM_ORG_ID, SERVICE_ORG_ID)
        .getCallsAPI(USER_ID)
          .create(IM_CALLS_ID);

Set&lt;CALLS_INCLUDE&gt; includeSet = Set.of(CALLS_INCLUDE.participants);
IMCalls call = ZohoIM.getInstance(IM_ORG_ID, SERVICE_ORG_ID)
        .getCallsAPI(USER_ID)
          .create(IM_CALLS_ID, includeSet);
```

### Get Calls List

```
IMCallsQueryBuilder queryBuilder = new IMCallsQueryBuilder();

queryBuilder.range(0, 10);

Set&lt;IMCallsStatus&gt; statusSet = new HashSet&lt;&gt;();
statusSet.add(IMCallsStatus.IN_PROGRESS);
queryBuilder.setCallsStatus(statusSet);

IMEntityList&lt;IMCalls&gt; calls = ZohoIM.getInstance(IM_ORG_ID, SERVICE_ORG_ID)
        .getCallsAPI(USER_ID)
          .getList(queryBuilder);
```
