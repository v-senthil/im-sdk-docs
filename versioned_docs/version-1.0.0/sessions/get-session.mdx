### Get a Session

```
IMSession session = ZohoIM.getInstance(IM_ORG_ID, SERVICE_ORG_ID)
        .getSessionAPI(USER_ID)
        .get(IM_SESSION_ID);
```

Possible Assignee Filter

1.  MINE
2.  UNASSIGNED
3.  NONE

```
IMSessionQueryBuilder queryBuilder = new IMSession.IMSessionQueryBuilder();

//When Assignee Filter is not given, AssigneeFilter.MINE will be considered as default.
queryBuilder.assigneeFilter(IMSessionQueryBuilder.AssigneeFilter.NONE);
queryBuilder.channelId(2490000000002145L); //Optional

//Get sessions from IM
IMEntityList&lt;IMSession&gt; sessionEntity = ZohoIM.getInstance(IM_ORG_ID, SERVICE_ORG_ID)
            .getSessionAPI(USER_ID)
            .getSessions(queryBuilder);
List&lt;IMSession&gt; sessionsList = sessionEntity.getEntities();
```
